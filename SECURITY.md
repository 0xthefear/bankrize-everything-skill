# Security & Scope

## Read-only by design

Bankrize Everything is a read-only token intelligence tool. It does **not**:

- swap, buy, or sell tokens
- transfer funds
- approve allowances
- claim, mint, or deploy
- place bets or automations
- sign messages or transactions

It only reads public market and activity data for Base tokens and presents a report.

## Not financial advice

The Trust Score and risk indicators are heuristics to help you research faster. They are not financial advice and are not a guarantee of safety. Always do your own research before interacting with any token.

## No false claims

Bankrize never claims liquidity is locked, or that a creator holds or sold, unless the underlying data confirms it. If a field can't be confirmed, it isn't shown.

## Reporting an issue

Found a bug or a data accuracy problem? Open an issue using the bug report template. For anything sensitive, please describe the problem without sharing private keys, seed phrases, or other secrets — Bankrize never needs them.
