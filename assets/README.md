# Assets

This folder holds images used in the project README and docs (app screenshots, GIFs, badges).

## Suggested files

- `dashboard.png` — the redesigned Bankrize dashboard (single-column, glassmorphism, aurora animation)
- `report-example.png` — a sample token report
- `trust-score.png` — a close-up of the trust score and risk indicators

## How to use in the README

Once an image is added here, reference it like:

```md
![Bankrize dashboard](./assets/dashboard.png)
```

Keep images focused on Bankrize Everything only.
