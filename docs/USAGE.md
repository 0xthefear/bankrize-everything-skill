# Usage

## In the app

1. Open the Bankrize app.
2. Paste a Base ticker or 0x contract address into the search box.
3. Hit Bankrize.
4. The active token auto-refreshes every 30 seconds.

## In chat

- "bankrize this token: 0x..."
- "use bankrize to analyze this Base token: 0x..."
- "give me a Bankrize report for 0x..."
- "Bankrize on Base: 0x..."

## On X / Twitter

For X replies, the first line is the bnkrize line and the last line is the data credit:

```
$bnkrize LFI

Price: $...
Market Cap: $...
Liquidity: $...
24h Volume: $...
1h: ...
6h: ...
Flow: ...
Trust Score: ../100
Risk: ...

Data from @bankrize
```

Rules:
- Don't add `$` before the token unless the user did.
- Never start with "Bankrized on Base" or "Bankrize Everything Report".
- Nothing comes after the final "Data from @bankrize" line.
