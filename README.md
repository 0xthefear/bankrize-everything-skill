# Bankrize Everything Skill

Bankrize Everything is a Bankr skill for analyzing Base tokens using the public Bankrize Everything app.

Public app:

[/u/0xc027ecbd0fa38f667a25550aae56b14c73931d5a/apps/bankrize-everything
](https://bankr.bot/u/0xc027ecbd0fa38f667a25550aae56b14c73931d5a/apps/bankrize-everything)
## What it does

Bankrize helps users analyze Base tokens with:

- live 5-minute trading activity
- live 1-hour trading activity
- total volume
- trade count
- buy vs sell flow
- average trade size
- largest recent trade
- near real-time activity feed
- liquidity
- market cap
- token age
- trust score
- risk indicators
- website and social links when available

## What it does not do

This skill does not trade.

It does not buy, sell, swap, transfer, approve, sign, claim, mint, deploy, or automate transactions.

It is read-only.

It is Base-only.

## Install

In Bankr, install this skill by giving Bankr the GitHub repo URL.

Example

install this skill: https://github.com/YOUR_USERNAME/bankrize-everything-skill

Replace YOUR_USERNAME with your GitHub username.

## Usage

After installing, users can say:

use bankrize to analyze this Base token: 0x...

or:

bankrize this token: 0x...

or:

give me a Bankrize report for 0x...
